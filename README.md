# 🧪 Chemical Symphony (化学交响乐)

将您的化学实验数据（如光谱、色谱、动态曲线等）转化为动听的钢琴交响乐。

## ✨ 功能特点

*   **多格式支持**: 支持上传 CSV, Excel (.xlsx/.xls), TXT 等常见实验数据格式。
*   **智能解析**: 自动识别时间轴 (X轴) 和数值轴 (Y轴)，无需复杂配置。
*   **情感化演奏**:
    *   **自动变速**: 根据数据变化的激烈程度，自动调整演奏速率（平缓乐段 vs 高潮乐段）。
    *   **古典风格**: 采用纯正的三角钢琴音色，配合左手自动分解和弦 (Arpeggios) 伴奏。
    *   **去噪处理**: 优化的断奏 (Staccato) 处理，消除机械合成音的拖沓感。
*   **交互式可视化**: 使用 Plotly 展示数据曲线，并在图上标记出音乐的强拍位置。
*   **即时预览与下载**: 支持在线试听合成音频，并可下载标准 MIDI 文件导入宿主软件 (DAW) 进行二次创作。
